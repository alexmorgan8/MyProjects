This project features a sample application showing one possible way to load images asynchronously in a UITableView, using Apple's Reachability class and the <a href="http://allseeing-i.com/ASIHTTPRequest">ASIHTTPRequest framework</a>. It also features a simple Core Animation effect similar to the one featured in the iTunes iPhone application.
